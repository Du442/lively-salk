# Portfolio Website

Portfólio pessoal desenvolvido para apresentar o perfil profissional de Lucas Rezende, suas habilidades, projetos recentes, trajetória profissional e informações de contato.

O projeto foi construído com HTML, CSS e JavaScript puro, sem dependência de frameworks, com foco em layout responsivo, navegação fluida e interações simples para melhorar a experiência do usuário.

## Funcionalidades

- Navbar fixa com mudança de estilo ao rolar a página.
- Menu mobile responsivo.
- Destaque automático do link ativo conforme a seção visível.
- Seção inicial com apresentação profissional e chamada para contato.
- Área "Sobre Mim" com informações pessoais e estatísticas.
- Seção de habilidades com barras animadas ao entrar na tela.
- Filtro de projetos por categoria.
- Linha do tempo com experiência profissional.
- Formulário de contato com simulação de envio.
- Rodapé com links para redes sociais.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Como Executar

Como o projeto é estático, não é necessário instalar dependências.

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Para uma experiência melhor durante o desenvolvimento, use uma extensão como Live Server.

## Principais Arquivos

- `index.html`: contém a estrutura principal da página e as seções do portfólio.
- `styles.css`: define o layout, cores, responsividade, animações e aparência visual.
- `app.js`: controla as interações da página, como menu mobile, filtros, animações e formulário.
- `assets/`: armazena imagens usadas no perfil e nos cards de projetos.

## Personalização

Para adaptar o portfólio, edite os textos, links, imagens e informações pessoais no arquivo `index.html`.

As cores, espaçamentos e estilos visuais podem ser alterados no arquivo `styles.css`.

As interações da página, como filtros e comportamento do menu, ficam concentradas no arquivo `app.js`.

## Observação

O formulário de contato não envia mensagens reais. Ele apenas simula o envio no navegador. Para torná-lo funcional, será necessário integrar um serviço de backend, API externa ou ferramenta de envio de e-mails.
